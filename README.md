# git-sample 
<p>생성,수정 파일을 새로운 git 레파지토리에 업로드 할 경우</p>
git init
git remote add origin
git branch -M main
git add . or git add 파일명
git push origin main

<p>깃허브에서 내려받기</p>
git clone 레파지토리 주소

<p>깃허브 branch 알기</p>
확인 git branch
생성 git branch 브랜치 명
브랜치바꿈 git checkout 브랜치 명

<p>remote</p>
git remote
git remote remove origin
